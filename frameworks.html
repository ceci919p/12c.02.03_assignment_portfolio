Choosing the framework

The framework used to solve this assignment was Pico as it seemed straightforward. This made me wonder how many options and custom settings the framework could offer.

Layout

The layout of the page is made by using a main tag with the class “container”. This automatically centers the content inside the main tag. To make the grid of the header I made a div with the class “grid” containing the text and image. This automatically made a two-column grid. This is the principle for all the grid columns on the page.

Semantic tags

By using semantic tags like article and form Pico automatically styles and makes the content responsive. This is how the responsive cards and contact form was created.

Customization

The website is the dark theme instead of light theme. The accent color is changed from the default blue to a bright pink. This was customized by copying the default CSS styling from Pico itself and overrule the default styling by giving it new values. This can be seen in the CTA buttons, links and the searchbar.
