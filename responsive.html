Planning the layout

Before coding the layout of the website, a little planning had to be done. The first thing was to observe the template in the assignment and note whether a section was layout or a component. All that was noted as layout was coded using Grid while all that was noted as components was coded using Flexbox. This helped the code process.

Layout and components

The advantages section in the template had a list of cards, like the one in the Fonts and UI Cards assignment. Therefor the cards were placed within the grid and the text contents of the cards was placed next to the image using flexbox. The flex direction in this assignment had to be column instead of row, but the principle was the same. The grid layout is only used on tablets and computers while most of the flex was used in all different screen sizes.

Sizing elements

To make the website even more responsive and accessible rem unit was used when sizing elements. If necessary, typography and images were adjusted depending on the screen size to make the layout work across all devices. Mostly the spacing of letters and images was adjusted rather than the font sizes.
